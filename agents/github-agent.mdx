# GitHub Agent

The GitHubAgent provides comprehensive GitHub integration through the GitHub MCP Server, enabling repository management, issue tracking, and code operations.

## Overview

| Property | Value |
|----------|-------|
| **ID** | `github-agent` |
| **Default Model** | `gemini-2.5-flash` |
| **Provider** | Google |
| **MCP Server** | github-mcp-server |

## Available Tools

The GitHub MCP Server provides these tools:

| Tool | Description |
|------|-------------|
| `list_commits` | Get commit history for a repository |
| `get_commit` | Get details of a specific commit |
| `search_repositories` | Search for repositories |
| `search_code` | Search code across GitHub |
| `issue_read` | Read issue details and comments |
| `issue_write` | Create or update issues |
| `list_pull_requests` | List PRs in a repository |
| `pull_request_read` | Get PR details, diff, or files |
| `get_file_contents` | Read file content from a repo |
| `create_or_update_file` | Create or modify files |
| `create_branch` | Create a new branch |
| `fork_repository` | Fork a repository |

## Authentication

The GitHub Agent requires a GitHub personal access token with appropriate permissions:

```bash
GITHUB_TOKEN=ghp_xxxxxxxxxxxxxxxxxxxx
```

Required scopes depend on operations:
- `repo` - Full repository access
- `read:org` - Read organization data
- `read:user` - Read user profile

## Configuration

```typescript
export const GitHubAgentConfig = {
  user_metadata: {
    id: "github-agent",
    name: "GitHub Agent",
    short_description: "GitHub Agent powered by Llama via Groq",
    long_description: "Interact with GitHub repositories, issues, PRs, and more."
  },
  implementation_metadata: {
    type: "api",
    provider: "google",
    modelID: "gemini-2.5-flash",
    systemInstruction: githubAgentSystemPrompt,
    apiKey: process.env.GEMINI_API_KEY
  }
}
```
