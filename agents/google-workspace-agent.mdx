The GoogleWorkspaceAgent provides unified access to Google Workspace services, enabling email management, calendar scheduling, file operations, and document editing.

## Overview

| Property | Value |
|----------|-------|
| **ID** | `google-workspace-agent` |
| **Default Model** | `gemini-2.5-flash` |
| **Provider** | Google |
| **Integration** | Google APIs |

## Service-Specific Features

### Gmail
- Send and reply to emails
- Search inbox by criteria
- Manage labels and folders
- Read email content and attachments

### Calendar
- Create and update events
- Find available time slots
- Send meeting invitations
- Check schedule conflicts

### Drive
- List and search files
- Create folders
- Share files with permissions
- Upload and download content

### Docs, Sheets & Slides
- Create new documents
- Edit existing content
- Format and style
- Collaborative editing support

## Authentication

The Google Workspace Agent requires OAuth 2.0 credentials:

```bash
GOOGLE_CLIENT_ID=your-client-id.apps.googleusercontent.com
GOOGLE_CLIENT_SECRET=your-client-secret
```

Users must authenticate through the OAuth flow to grant access to their Google Workspace data.

## Configuration

```typescript
export const GoogleWorkspaceAgentConfig = {
  user_metadata: {
    id: "google-workspace-agent",
    name: "Google Workspace Agent",
    short_description: "Manage Gmail, Calendar, Drive, Docs, Sheets, and Slides",
    long_description: "A unified agent for all Google Workspace services..."
  },
  implementation_metadata: {
    type: "api",
    provider: "google",
    modelID: "gemini-2.5-flash",
    systemInstruction: googleWorkspaceAgentSystemPrompt,
    apiKey: process.env.GEMINI_API_KEY
  }
}
```
