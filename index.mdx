## Key Features

<AccordionGroup>
  <Accordion title="Multi-Agent Orchestration">
    The MainAgent automatically delegates tasks to specialized sub-agents based on the user's intent. No manual routing required.
  </Accordion>
  <Accordion title="Specialized Agents">
    Purpose-built agents for GitHub, Google Workspace, web content extraction, HuggingFace, and UI customization.
  </Accordion>
  <Accordion title="MCP Integration">
    Model Context Protocol (MCP) servers provide standardized tool interfaces for external services.
  </Accordion>
  <Accordion title="Flexible LLM Support">
    Configure different LLM providers (Google, OpenAI, Anthropic, Groq) per agent.
  </Accordion>
</AccordionGroup>

## Available Agents

| Agent | Description |
|-------|-------------|
| **MainAgent** | Orchestrator that routes requests to specialized agents |
| **GitHubAgent** | Interact with repositories, issues, PRs, and commits |
| **GoogleWorkspaceAgent** | Manage Gmail, Calendar, Drive, Docs, Sheets, Slides |
| **WebAgent** | Extract content and metadata from web pages |
| **HuggingFaceAgent** | Discover ML models, datasets, and papers |
| **FrontendAgent** | Customize UI themes and styling via natural language |
