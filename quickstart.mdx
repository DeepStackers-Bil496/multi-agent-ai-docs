## Installation

<Steps>
  <Step title="Clone the Repository">
    ```bash
    git clone https://github.com/DeepStackers-Bil496/multi-agent-ai-framework.git
    cd multi-agent-ai-framework
    ```
  </Step>
  
  <Step title="Install Vercel CLI">
    ```bash
    npm i -g vercel
    ```
  </Step>
  
  <Step title="Link to Vercel Project">
    Connect your local instance with Vercel and GitHub accounts:
    ```bash
    vercel link
    ```
    This creates a `.vercel` directory with project configuration.
  </Step>
  
  <Step title="Pull Environment Variables">
    Download your environment variables from Vercel:
    ```bash
    vercel env pull
    ```
  </Step>
  
  <Step title="Install Dependencies">
    ```bash
    pnpm install
    ```
  </Step>
  
  <Step title="Setup Database">
    Run database migrations:
    ```bash
    pnpm db:migrate
    ```
  </Step>
  
  <Step title="Start Development Server">
    ```bash
    pnpm dev
    ```
  </Step>
</Steps>

Your app should now be running on [localhost:3000](http://localhost:3000).

## Environment Variables

The following environment variables are required:

| Variable | Description |
|----------|-------------|
| `GEMINI_API_KEY` | Google Gemini API key |
| `POSTGRES_URL` | Vercel Postgres connection URL |
| `AUTH_SECRET` | NextAuth.js secret key |
| `GITHUB_TOKEN` | GitHub personal access token (optional) |
| `GOOGLE_CLIENT_ID` | Google OAuth client ID (optional) |
| `GOOGLE_CLIENT_SECRET` | Google OAuth client secret (optional) |
